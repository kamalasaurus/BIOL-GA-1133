1 0) Overview

We build geometry-first intuition: - Dot product = projection (signed length). - Outer product = project then stretch (rank-1 map). - Matrix multiplication = many project-and-stretch actions (sum of rank-1 tiles).

2 1) Dot product projection

For vectors \(u,v\in\mathbb{R}^n\), \[ u\cdot v = \sum_i u_i v_i = \|u\|\,\|v\|\cos\theta. \] The projection vector of \(u\) onto the line of \(v\) is \[ \mathrm{Proj}_v(u) = \frac{u\cdot v}{\|v\|^2}\, v. \]

vector_u <- c(3, 1)
vector_v <- c(2, 0)
dot_product <- sum(vector_u * vector_v)
projection_vector <- (dot_product / sum(vector_v^2)) * vector_v
magnitude_u <- sqrt(sum(vector_u^2))
magnitude_v <- sqrt(sum(vector_v^2))
angle_between_vectors <- acos(dot_product / (magnitude_u * magnitude_v)) * 180/pi
list(dot_product = dot_product, projection_vector = projection_vector, angle_degrees = angle_between_vectors)

3 2) Outer product project then stretch

For \(u\in\mathbb{R}^m, v\in\mathbb{R}^n\), the outer product \(uv^\top\) acts on any \(x\in\mathbb{R}^n\) as \[ (uv^\top) x = u\,(v^\top x). \] - First project: compute the scalar \(v^\top x\). - Then stretch along u by that scalar.

vector_u <- c(1, 2)
vector_v <- c(2, 1)
outer_product_matrix <- vector_u %o% vector_v  # outer product matrix
test_vectors <- list(c(3, 0), c(0, 3), c(2, 2))
transformation_results <- lapply(test_vectors, function(input_vector) {
  list(input = input_vector, 
       output = outer_product_matrix %*% input_vector, 
       projection_scalar = sum(vector_v * input_vector))
})

4 3) Matrix multiplication sum of many project-and-stretch actions

For conformable \(A\in\mathbb{R}^{m\times K}, B\in\mathbb{R}^{K\times n}\), \[ AB = \sum_{k=1}^K A_{:k} B_{k:}. \] Each term \(A_{:k}B_{k:}\) is an outer product: project along a row of \(B\), stretch along a column of \(A\).

matrix_A <- matrix(c(1, 2, 3, 4), nrow = 2, byrow = TRUE)
matrix_B <- matrix(c(5, 6, 7, 8), nrow = 2, byrow = TRUE)
sum_of_outer_products <- outer(matrix_A[, 1], matrix_B[1, ]) + outer(matrix_A[, 2], matrix_B[2, ])
matrix_multiplication_result <- matrix_A %*% matrix_B
matrices_are_equal <- all.equal(sum_of_outer_products, matrix_multiplication_result)
matrices_are_equal
